🏥 Healthcare Data Analysis using SQL

This project focuses on analyzing healthcare data using Structured Query Language (SQL) to generate meaningful insights from patient, visit, and treatment datasets. The objective is to support data-driven decision-making in hospital management and healthcare operations.

🔹 Project Overview

The database consists of multiple relational tables such as Patients, Visits, Treatments, and Departments. By applying SQL queries, the project extracts valuable information related to revenue generation, patient flow, treatment costs, and departmental performance.

🔹 Key Features

* Complex **JOIN operations** to combine multiple tables
* Use of **GROUP BY, HAVING, ORDER BY** for summarized insights
* Implementation of **aggregate functions** (SUM, COUNT, AVG, ROUND)
* **Subqueries and nested queries** for advanced data analysis
* Monthly visit trend analysis using **DATE functions**
* Revenue analysis by state and department
* Identification of top-performing departments
* Patient count and treatment distribution analysis

🔹 Insights Generated

* Total hospital revenue and revenue in millions
* Monthly and yearly patient visit trends
* Department-wise performance comparison
* State-wise revenue contribution
* Average treatment cost per visit

🔹 Tools & Skills Used

* MySQL / SQL Server
* Data Cleaning & Query Optimization
* Relational Database Concepts
* Analytical Thinking & Problem Solving

This project demonstrates strong SQL fundamentals, database management skills, and the ability to transform raw healthcare data into actionable business insights.
